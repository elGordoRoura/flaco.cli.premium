You have three options:

#### Option B: OpenAI GPT

### Step 3: Get Your API Key

#### For Anthropic Claude:
1. Go to [console.anthropic.com](https://console.anthropic.com/)
2. Sign up or log in
3. Navigate to **API Keys**
4. Create a new API key
5. Copy and paste it into Flaco

#### For OpenAI:
1. Go to [platform.openai.com/api-keys](https://platform.openai.com/api-keys)
2. Sign up or log in
3. Create a new API key
4. Copy and paste it into Flaco

#### For Local AI:
1. Install [Ollama](https://ollama.ai) or [LM Studio](https://lmstudio.ai)
2. Start the local server
3. Enter the endpoint URL (usually `http://localhost:11434`)

### Step 4: Choose Your Model

Select which AI model you want to use:
- **Claude Sonnet 4** (Recommended) - Best balance of speed and quality
- **Claude 3.5 Sonnet** - Fast and capable
- **GPT-4 Turbo** - OpenAI's latest
- **GPT-4** - Classic reliable model

## Tips & Tricks

### 1. Be Specific
Instead of: "Help me with my code"
Try: "Review this authentication function and suggest improvements"

### 2. Use Context
Flaco works best when you provide context:
```
I'm building a React app with TypeScript.
Can you help me create a custom hook for API calls?
```

### 3. Iterative Refinement
Don't be afraid to ask follow-up questions:
```
User: Create a login form
Flaco: [creates basic form]
User: Add validation and error handling
Flaco: [improves the form]
```

### 4. Command Mode
Use `/` commands for special actions:
- `/scan` - Scan current project
- `/git status` - Check git status
- `/stats` - View statistics

## Troubleshooting

### "Error: API key not initialized"
1. Open Settings (âš™ï¸)
2. Re-enter your API key
3. Make sure there are no extra spaces

### "Error: Invalid API key"
1. Check that you copied the full key
2. Verify the key is for the correct provider (Anthropic vs OpenAI)
3. Generate a new key if needed

### Slow Responses
- Try a faster model (GPT-3.5 Turbo or Claude Sonnet)
- Check your internet connection
- Consider using a local AI model

### Can't Send Messages
1. Make sure you completed the first-run setup
2. Check that your API key is valid
3. Restart the app

## Privacy & Security

### Your Data
- **API Keys** are stored encrypted on your local machine
- **Conversations** stay local (not uploaded to Flaco servers)
- **API Providers** (Anthropic/OpenAI) process your messages according to their policies

### Security Best Practices
- Never share your API keys
- Rotate API keys regularly
- Use local AI for sensitive projects
- Review provider privacy policies
